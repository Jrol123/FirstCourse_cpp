Подробное описание

template<typename type>
struct DSF< type >

Класс DSF (Disjoint Set Forest) для упорядочивания множеств.

DSF позволяет объединять множества и определять, в каком множестве находится элемент.

Параметры шаблона
typeТип элементов множества.
